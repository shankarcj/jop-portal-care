# Jop Poratal nodejs Application

## Technologies
* Node.js
* Express
* Express Messages, Session, Connect Flash & Validation
* MongoDB & Mongoose
* Ejs templating
* Passport.js Authentication
* BCrypt Hashing

### Version
2.0.0

## Usage


### Installation

Install the dependencies

```sh
$ npm install
```
Run app

```sh
$ npm start
```
